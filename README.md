### 🔵 [Windows Log Analysis Lab](./lab-002-windows-log-analysis.md)
Configured Windows 11 audit policies via secpol.msc 
and deliberately generated real security events. 
Analyzed Event IDs 4624, 4625, 4720, 4732, and 4726

### 🔵 [Wireshark DNS Traffic Analysis](./lab-003-wireshark-dns-analysis.md)

Identified DNS query and response patterns, analyzed 
port 53 UDP traffic, and investigated simulated DNS 
tunneling behavior using Base64 encoded queries.

against a Windows 11 VM. Detected the attack using
